## Mapping DNA → UTF-32

```
A (Adenine)  → ∀  (always pairs with T)
T (Thymine)  → 𝑡  (time, pairs with A)
G (Guanine)  → ∇  (gradient, pairs with C)
C (Cytosine) → ◌  (complement, pairs with G)

Codon (3 bases) → 1 amino acid → 1 UTF-32 node
Protein         → function (∫, ∇, ∪, ...)
Gene            → module / function definition
Chromosome      → library / package
Genome          → complete program
```

## Bài học cho SILK

```
DNA đã làm điều SILK muốn làm:
  - 4 ký tự (A,T,G,C)
  - Encode toàn bộ thông tin sự sống
  - Universal: mọi sinh vật dùng cùng code
  - Self-executing: DNA tự chạy (qua ribosome)
  - Self-replicating: code tự sao chép

SILK học từ DNA:
  - Ít ký tự gốc, nhiều combination
  - Universal across "species" (languages)
  - Self-describing
```
