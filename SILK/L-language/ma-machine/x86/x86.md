# x86 -- Intel/AMD Architecture

born:      1978 (Intel 8086)
type:      CISC — Complex Instruction Set Computing
dominant:  desktop, laptop, server (>90% market)
variants:  x86 (32-bit), x86-64/AMD64 (64-bit)

## CISC philosophy

    Many complex instructions → compiler does less work
    x86: MOV EAX, [EBX + ECX*4 + 8]  -- 1 instruction!
    vs ARM RISC: needs 3+ instructions for same

## Instructions → symbols

    MOV, LOAD, STORE  -- memory access              (∂)
    ADD, SUB, MUL     -- arithmetic                 (∑)
    AND, OR, XOR      -- bitwise set operations     (∪∩)
    CMP, JMP, JE, JNE -- conditional branch         (∃∀)
    CALL, RET         -- function application       (→)
    PUSH, POP         -- stack operations           (∂t)
    LOOP, DEC+JNZ     -- iteration                  (♻)
    SIMD SSE/AVX      -- parallel for-all           (∀)

## x86-64 calling convention (System V ABI)

    Arguments: RDI, RSI, RDX, RCX, R8, R9
    Return:    RAX
    Preserved: RBX, RBP, R12-R15

    ; float sdf_sphere(p: Vec3, c: Vec3, r: float)
    sdf_sphere:
        SUBSS  XMM0, XMM3    ; px - cx
        SUBSS  XMM1, XMM4    ; py - cy
        SUBSS  XMM2, XMM5    ; pz - cz
        MULSS  XMM0, XMM0
        MULSS  XMM1, XMM1
        MULSS  XMM2, XMM2
        ADDSS  XMM0, XMM1    ; sum
        ADDSS  XMM0, XMM2
        SQRTSS XMM0, XMM0    ; sqrt of sum
        SUBSS  XMM0, XMM6    ; - r
        RET

## Timeline

    1978: 8086 -- 16-bit
    1985: 80386 -- 32-bit protected mode
    2003: AMD64 -- 64-bit extension
    2024: Used in majority of servers and desktops
