## Nguyên lý

Mỗi **lá** = 1 khái niệm, được mô tả bằng:
- `symbol` — ký tự UTF-32
- `semantic` — nghĩa thuần túy
- `projections` — cách viết trong mỗi ngôn ngữ
- `relations` — quan hệ với khái niệm khác
